<h1 align="center">MULTI AUTHENTICATION<h1>

 Process
- composer require laravel/ui 
- php artisan ui bootstrap --auth 
- npm install
- npm run dev
- Add this "$table->boolean('user_type')->default(0);" in user migration file.
- php artisan make:middleware IsAdmin
